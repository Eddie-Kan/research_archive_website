## CsPbI3两步旋涂合成协议

### 材料

- CsI（99.999%，Sigma-Aldrich）
- PbI2（99.99%，TCI）
- DMF（无水，Sigma-Aldrich）
- DMSO（无水，Sigma-Aldrich）

### 第一步：PbI2薄膜沉积

1. 将PbI2溶解在DMF:DMSO（9:1 v/v）中，浓度为1.0 M
2. 在70°C下搅拌2小时至完全溶解
3. 在清洁的ITO基底上以3000 rpm旋涂30秒
4. 在70°C下退火1分钟

### 第二步：CsI处理

1. 在异丙醇中配制1.5 mg/mL CsI溶液
2. 在PbI2薄膜上以2000 rpm旋涂30秒
3. 在氮气氛围中150°C退火10分钟

### 常见失败模式

- **转化不完全**：CsI浓度过低或退火温度不足
- **针孔**：旋涂速度过高或基底未正确清洁
- **相不稳定**：退火期间环境湿度超过30%

### 提示

[[mat-perovskite-abx3|钙钛矿]]相对湿度敏感。始终在手套箱或RH < 20%的干燥室中操作。
